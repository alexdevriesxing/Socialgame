# Changelog

## 0.1.0 — Vertical Slice

- Added complete canvas-based 2D game shell.
- Added original pixel-art school map, character sprites, portraits, icons and key art.
- Added separate boys' and girls' ladders.
- Added bell-driven class schedule, attendance, lateness, demerits and detention.
- Added ten student NPCs, four teachers and four selectable clubs.
- Added rich dialogue choices, class prompts, missions, events and family-friendly conflicts.
- Added daily summaries, monthly finals, 48-month progression structure and prom ending.
- Added local save/continue, touch controls, audio and static-host deployment files.
