# Walkable World Controls

The academy, bedroom and all thirteen district destinations use the same direct 2D movement model.

- **WASD / Arrow keys:** move.
- **E / Space:** interact with the nearest classmate, prop, shop, activity or exit.
- **X:** open the illustrated district map.
- **B:** enter the bedroom.
- **U:** open the collection book while retaining the current location.
- **Escape:** return to Sakura Crest Academy from a district area.

Each map has authored collision geometry, camera containment, a valid player spawn, a district-map exit and an academy return point. Visiting classmates wander locally and can trigger their off-campus friendship scene or a contextual conversation.
