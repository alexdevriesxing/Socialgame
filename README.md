# Sakura Crest: Social Summit — Cloudflare Release

This branch is a prebuilt static release of the Anime Campaign Pass v0.4. It includes the complete four-year / 48-chapter campaign framework and generates its crisp anime sprites, portraits, seasonal maps, event art, UI and tiles as scalable SVG atlases in the browser.

## Cloudflare Pages

- Framework preset: **None**
- Build command: leave blank
- Build output directory: **/** or **.**
- Environment variables: none

The repository root is the deployable site. No Workers, package installation, database, external image host or secrets are required.

## Controls

WASD / arrows to move, E or Space to interact, R for rankings, Q for schedule, M for sound and Escape to pause. Touch controls are built in.

## Scope

This is a deployable campaign beta with all 48 monthly chapters, the complete timetable/ranking/club/relationship loop and prom ending. Further commercial polish can add more bespoke daily scenes, route illustrations, animations and mini-games.
